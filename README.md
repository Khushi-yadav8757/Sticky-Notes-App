# Sticky Notes Web App

A modern and responsive Sticky Notes application built using **HTML, CSS, and Vanilla JavaScript**.  
This app allows users to create, edit, delete, search, and organize notes directly in the browser.

All notes are stored using the **LocalStorage API**, which ensures that notes remain available even after refreshing the page.

## Features

- Create new sticky notes
- Edit note title and description
- Delete notes
- Pin important notes
- Search notes
- Multiple note colors
- Dark mode support
- Drag and drop notes
- Export notes as JSON
- Responsive design for desktop and mobile

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## Learning Objectives

This project demonstrates practical front-end development skills such as:

- DOM Manipulation
- LocalStorage data persistence
- Responsive UI design
- JavaScript event handling
- Drag and Drop API
